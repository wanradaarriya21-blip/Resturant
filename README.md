# Siam Bistro - เว็บแอปพลิเคชันสั่งอาหารออนไลน์ระดับพรีเมียม

ระบบสั่งอาหารออนไลน์แบบ Single Page Application (SPA) ที่สวยงาม ใช้งานง่าย โหลดเร็ว รองรับการเชื่อมต่อแบบเรียลไทม์กับฐานข้อมูล Supabase หรือสามารถใช้งานแบบทดลองออฟไลน์ได้ทันที พร้อมระบบการออกใบเสร็จรับเงินในรูปแบบธีมสว่าง (Light Mode Receipt) ที่ออกแบบมาเพื่อการจัดพิมพ์อย่างชัดเจน

---

## ฟีเจอร์เด่น
1. **ดีไซน์ระดับพรีเมียม**: ใช้การคุมโทนสีอบอุ่น (Warm Colors) พร้อมการทำเงาแบบ Glassmorphism และ Micro-animations ที่ลื่นไหล
2. **รองรับฐานข้อมูล Supabase**: สามารถส่งรายการสินค้าที่สั่งซื้อเก็บเข้าฐานข้อมูลจริงใน Cloud ได้อย่างปลอดภัย
3. **ระบบสับเปลี่ยนโหมด (Hybrid Mode)**: สามารถรันในโหมดจำลอง (Mock Mode) เพื่อทดสอบฟังก์ชันสั่งอาหารผ่าน LocalStorage ได้ทันทีโดยไม่ต้องตั้งค่าฐานข้อมูลล่วงหน้า
4. **ใบเสร็จสไตล์กระดาษใบเสร็จ Light Mode**: ใบเสร็จกระดาษสีขาว มีขอบหยักและบาร์โค้ดจำลองที่คมชัด และรองรับระบบสั่งพิมพ์ (Print View) ของเว็บบราวเซอร์โดยจะตัดส่วนนำทางออกไปและพิมพ์เฉพาะใบเสร็จเท่านั้น

---

## โครงสร้างไฟล์
* `index.html` : หน้าหลักและโครงสร้างหน้าเว็บสั่งอาหาร
* `styles.css` : สไตล์ของแอปพลิเคชันและการแสดงผลใบเสร็จสำหรับสั่งพิมพ์
* `app.js` : ตรรกะการทำงาน ตะกร้าสินค้า และการเชื่อมต่อ Supabase
* `supabase_schema.sql` : รหัส SQL สำหรับนำไปรันเพื่อตั้งค่าฐานข้อมูลใน Supabase
* `README.md` : ไฟล์คู่มือการใช้งานนี้

---

## ขั้นตอนการติดตั้งและเชื่อมต่อกับ Supabase

เพื่อใช้ระบบเก็บข้อมูลผ่าน Supabase จริง ให้ทำตามขั้นตอนดังนี้:

### ขั้นตอนที่ 1: ตั้งค่าฐานข้อมูล Supabase
1. เข้าไปที่เว็บ [Supabase.com](https://supabase.com) จากนั้นเข้าสู่ระบบและสร้างโปรเจกต์ใหม่ (New Project)
2. เมื่อโปรเจกต์พร้อมแล้ว ให้ไปที่เมนู **SQL Editor** ในแถบเมนูด้านซ้าย
3. กดปุ่ม **New Query**
4. เปิดไฟล์ `supabase_schema.sql` จากโครงการนี้ คัดลอกเนื้อหาทั้งหมดไปวางในช่อง SQL Editor จากนั้นกดปุ่ม **Run**
5. ตรวจสอบว่าตาราง `menu_items`, `orders` และ `order_items` ถูกสร้างเรียบร้อยแล้ว

### ขั้นตอนที่ 2: คัดลอกค่าการเชื่อมต่อ API
1. ไปที่เมนู **Project Settings** (ปุ่มรูปฟันเฟืองด้านซ้ายล่าง)
2. เลือกแถบ **API**
3. คัดลอกข้อมูลในช่อง:
   * **Project URL** (เช่น `https://your-project.supabase.co`)
   * **Project API keys** ในส่วนของ **anon public**

### ขั้นตอนที่ 3: ตั้งค่าในเว็บแอปพลิเคชัน
1. เปิดไฟล์ `index.html` ในเว็บบราวเซอร์ใดก็ได้
2. สังเกตที่ปุ่มไอคอนรูปฐานข้อมูล (Database) มุมบนขวา หรือกดที่แถบแจ้งเตือนสีเหลืองด้านบน
3. กรอก **Supabase URL** และ **API Key (anon)** ที่คุณคัดลอกมาลงในช่องข้อความ
4. กดปุ่ม **เชื่อมต่อฐานข้อมูล**
5. เว็บไซต์จะทำการดาวน์โหลดและซิงค์ข้อมูลกับ Supabase ของคุณทันที และหากตารางเมนูอาหารของคุณยังว่างเปล่า ระบบจะทำการสร้างข้อมูลเมนูอาหารเริ่มต้น (Seed) ให้โดยอัตโนมัติ!

## วิธีการใช้งานแอปพลิเคชัน

1. **เลือกอาหาร**: กดสลับประเภทอาหารที่ต้องการด้านบน จากนั้นคลิก **"เพิ่มลงตะกร้า"** เพื่อเพิ่มสินค้า หรือปรับเพิ่ม/ลดจำนวน
2. **ตรวจสอบตะกร้า**: คลิกปุ่ม **"ตะกร้าสินค้า"** มุมบนขวาเพื่อเลื่อนดูรายการ ยอดรวม และอัตราภาษี 7%
3. **ส่งสั่งซื้อ**: กรอกชื่อผู้สั่งอาหาร เบอร์โทรศัพท์ และเลขโต๊ะอาหารของคุณ จากนั้นกดปุ่ม **"ยืนยันการสั่งอาหารและชำระเงิน"**
4. **พิมพ์ใบเสร็จ**: เมื่อกดส่งซื้อแล้ว ระบบจะแสดงหน้าสั่งซื้อสำเร็จพร้อมใบเสร็จสไตล์ Light Mode คุณสามารถกดปุ่ม **"พิมพ์ใบเสร็จ"** เพื่อสั่งพิมพ์กระดาษจริงหรือบันทึกเป็น PDF ได้ทันที

*หมายเหตุ: ในขั้นตอนกดพิมพ์ใบเสร็จ บราวเซอร์จะแสดงผลพรีวิวเฉพาะตัวใบเสร็จรับเงินสีขาวเท่านั้น เพื่อความประหยัดหมึกพิมพ์และความสวยงามตามมาตรฐาน*
